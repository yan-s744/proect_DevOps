# Развертывание ВМ в Яндекс облаке с помощью Terraform и Ansible.
Развернул с помощью Terraform три виртуальные машины с доступом по SSH.
Ansible также подключается к этим машинам с помощью указанного SSH-ключа.
У Ansible три роли:  
1) изменение имени машины и установка стандартных — net-tools, dnsutils, wget, curl, mc, rsync — пакетов; 
2) Настройка проксирующего Nginx;
3) Настройка отдающего Nginx.
