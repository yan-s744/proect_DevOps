Terraform нужен для разворачивания, изменения, удаления и резервного копирования виртуальных машин и контейнеров в облачных инфраструктурах, системах виртуализации или контейнеризации. Terraform написан на Go.
- Для описания переменных в Terraform обязательны два файла: variables.tf и terraform.tfvars.
- Конфигурационными файлами являются providers.tf, main.tf.
